# MetaMirror

MetaMirror is an AI-powered mental well-being platform that adapts therapy sessions to a user's emotional state. It combines immersive React visuals, FastAPI, MongoDB, emotion tracking, and Gemini chat to deliver personalized reflection, coping support, and session analytics.

## Tech Stack

- React, CRACO, Tailwind CSS, Radix UI, Three.js, Recharts
- FastAPI, Uvicorn, Pydantic, MongoDB, Motor
- Google Gemini API
- Vercel deployments for frontend and backend API

## Status

The project is deployed successfully. Backend chat and database features require production environment variables such as `MONGO_URL` and `GEMINI_API_KEY`.
